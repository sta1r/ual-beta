**** Installation Instructions:

To enable cross domain access for asset files in IE8. 

1. Place the files in the assets/ folder on your own server

2. Update the HTML (lines 59 & 62) to point to respond.proxy.js and respond.proxy.gif on your server.


See https://github.com/scottjehl/Respond#cdnx-domain-setup for more information.
