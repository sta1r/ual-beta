A large but not exhaustive list of the key components, tools and services that have gone into the construction of beta.arts.ac.uk The tools we use will change and evolve over time so this list represents the state of things on August 7th 2013

Frontend:

  • HTML/CSS/JS - using HTML5 where appropriate, with a heavy focus on accessibility, and validating where we can
  • We are using ReSRC.it to handle our responsive images, and Polyfill to handle the media queries for the ReSRC.it
  • We use jquery as our primary javascript library
  • Video playback with the Nomensa Accessible Media Player
  • Backend admin systems make use of Gridset
  • We use SCSS and Compass, as seen in our frontend toolkit

Supporting Tools:

  • We keep on track and in touch using Trello, Basecamp and Google Docs
  • Github helps us manage and discuss our code
  • Zendesk keeps the feedback flowing
  • We use jekyllfor some of our prototyping
